#### 例子：根据脚手架创建一个服务

```shell
mvn archetype:generate -DgroupId=cn.chenyunlong -DartifactId=qing-domain-message -Dversion=0.0.1-SNAPSHOT -Dpackage=cn.chenyunlong.qing.message -DappName=qing-message-center -DarchetypeArtifactId=qing-service-archetype -DarchetypeGroupId=cn.chenyunlong -DarchetypeVersion=0.0.1-SNAPSHOT
```
